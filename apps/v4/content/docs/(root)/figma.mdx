## 付费资源

- [shadcn/ui kit](https://shadcndesign.com)，作者 [Matt Wierzbicki](https://x.com/matsugfx) —— 一套高质量且持续更新的 Figma UI Kit，全面兼容 shadcn/ui，并针对设计到开发的协作流程做了优化。
- [Shadcraft UI Kit](https://shadcraft.com) —— shadcn 兼容的高级套件，内置 [tweakcn](https://tweakcn.com) 提供的即时主题切换、专业组件与模板库，以及对 shadcn 组件与区块的完整覆盖。

## 免费资源

- [shadcn/ui design system](https://www.figma.com/community/file/1203061493325953101)，作者 [Pietro Schirano](https://twitter.com/skirano) —— shadcn/ui 的设计伴侣，每个组件都精心制作，以完美匹配代码实现。
- [Obra shadcn/ui](https://www.figma.com/community/file/1514746685758799870/obra-shadcn-ui)，作者 [Obra Studio](https://obra.studio/) —— 遵循 shadcn 理念打造的精细套件，跟随 v4 同步更新，并以 MIT 许可发布。
