## 用法

```tsx
import {
  ButtonGroup,
  ButtonGroupSeparator,
  ButtonGroupText,
} from "@/components/ui/button-group"
```

```tsx
<ButtonGroup>
  <Button>按钮 1</Button>
  <Button>按钮 2</Button>
</ButtonGroup>
```

## 无障碍

- `ButtonGroup` 组件的 `role` 属性被设置为 `group`。
- 使用 <Kbd>Tab</Kbd> 在组内按钮之间导航。
- 使用 `aria-label` 或 `aria-labelledby` 为按钮组提供名称。

```tsx showLineNumbers
<ButtonGroup aria-label="Button group">
  <Button>按钮 1</Button>
  <Button>按钮 2</Button>
</ButtonGroup>
```

## ButtonGroup 与 ToggleGroup

- 当你需要将执行动作的按钮归组时，使用 `ButtonGroup`。
- 需要将切换状态的按钮归组时，使用 `ToggleGroup`。

## API 参考

### ButtonGroup

`ButtonGroup` 组件是一个容器，用来将相关按钮组合在一起并保持统一样式。

| Prop          | Type                         | Default        |
| ------------- | ---------------------------- | -------------- |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` |

```tsx
<ButtonGroup>
  <Button>按钮 1</Button>
  <Button>按钮 2</Button>
</ButtonGroup>
```

可以嵌套多个按钮组来创建带间距的复杂布局，详见 [嵌套](#嵌套) 示例。

```tsx
<ButtonGroup>
  <ButtonGroup />
  <ButtonGroup />
</ButtonGroup>
```

### ButtonGroupSeparator

`ButtonGroupSeparator` 组件用于在按钮组中划分视觉区域。

| Prop          | Type                         | Default      |
| ------------- | ---------------------------- | ------------ |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` |

```tsx
<ButtonGroup>
  <Button>按钮 1</Button>
  <ButtonGroupSeparator />
  <Button>按钮 2</Button>
</ButtonGroup>
```

### ButtonGroupText

用于在按钮组内显示文本。

| Prop      | Type      | Default |
| --------- | --------- | ------- |
| `asChild` | `boolean` | `false` |

```tsx
<ButtonGroup>
  <ButtonGroupText>Text</ButtonGroupText>
  <Button>Button</Button>
</ButtonGroup>
```

Use the `asChild` prop to render a custom component as the text, for example a label.

```tsx showLineNumbers
import { ButtonGroupText } from "@/components/ui/button-group"
import { Label } from "@/components/ui/label"

export function ButtonGroupTextDemo() {
  return (
    <ButtonGroup>
      <ButtonGroupText asChild>
        <Label htmlFor="name">Text</Label>
      </ButtonGroupText>
      <Input placeholder="Type something here..." id="name" />
    </ButtonGroup>
  )
}
```
