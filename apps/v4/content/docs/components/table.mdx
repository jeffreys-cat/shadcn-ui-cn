## 用法

```tsx showLineNumbers

  Table,
  TableBody,
  TableCaption,
  TableCell,
  TableHead,
  TableHeader,
  TableRow,

```

```tsx showLineNumbers

  <TableCaption>最近的账单列表。</TableCaption>
```

## 数据表格

你可以使用 `<Table />` 组件构建更复杂的数据表。结合 [@tanstack/react-table](https://tanstack.com/table/v8) 可以实现排序、筛选和分页等功能。

更多信息请参阅 [数据表格](/docs/components/data-table) 文档。

你还可以在 [Tasks](/examples/tasks) 示例中查看一个数据表格的完整实现。
