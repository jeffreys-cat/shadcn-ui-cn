### 间距

通过在 `<CarouselItem />` 上设置 `pl-[VALUE]`，并在 `<CarouselContent />` 上设置负的 `-ml-[VALUE]` 来控制条目间距。

<Callout className="mt-6">
  **原因：** 虽然可以在 `<CarouselContent />` 上使用 `gap` 或 `grid` 布局，但需要大量计算才能获得合适的间距。相比之下，使用 `pl-[VALUE]` 与 `-ml-[VALUE]` 更直观。

如有需要，可在自己的项目中自由调整。

</Callout>

## Options

通过 `opts` 属性向轮播传入配置项。更多说明见 [Embla Carousel 文档](https://www.embla-carousel.com/api/options/)。

```tsx showLineNumbers {2-5}
<Carousel
  opts={{
    align: "start",
    loop: true,
  }}
>
  <CarouselContent>
    <CarouselItem>...</CarouselItem>
    <CarouselItem>...</CarouselItem>
    <CarouselItem>...</CarouselItem>
  </CarouselContent>
</Carousel>
```

## API

使用状态与 `setApi` 属性获取轮播 API 实例。

<ComponentPreview
  name="carousel-api"
  title="轮播"
  description="带幻灯片计数器的轮播。"
/>

```tsx showLineNumbers {1,4,22}
import { type CarouselApi } from "@/components/ui/carousel"

export function Example() {
  const [api, setApi] = React.useState<CarouselApi>()
  const [current, setCurrent] = React.useState(0)
  const [count, setCount] = React.useState(0)

  React.useEffect(() => {
    if (!api) {
      return
    }

    setCount(api.scrollSnapList().length)
    setCurrent(api.selectedScrollSnap() + 1)

    api.on("select", () => {
      setCurrent(api.selectedScrollSnap() + 1)
    })
  }, [api])

  return (
    <Carousel setApi={setApi}>
      <CarouselContent>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
      </CarouselContent>
    </Carousel>
  )
}
```

## 事件

可以借助 `setApi` 返回的实例监听事件。

```tsx showLineNumbers {1,4-14,16}
import { type CarouselApi } from "@/components/ui/carousel"

export function Example() {
  const [api, setApi] = React.useState<CarouselApi>()

  React.useEffect(() => {
    if (!api) {
      return
    }

    api.on("select", () => {
      // Do something on select.
    })
  }, [api])

  return (
    <Carousel setApi={setApi}>
      <CarouselContent>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
        <CarouselItem>...</CarouselItem>
      </CarouselContent>
    </Carousel>
  )
}
```

有关事件的更多信息，请查看 [Embla Carousel 文档](https://www.embla-carousel.com/api/events/)。

## 插件

可以通过 `plugins` 属性为轮播添加插件。

```ts showLineNumbers {1,6-10}
import Autoplay from "embla-carousel-autoplay"

export function Example() {
  return (
    <Carousel
      plugins={[
        Autoplay({
          delay: 2000,
        }),
      ]}
    >
      // ...
    </Carousel>
  )
}
```
